To get search points, you gotta be logged in properly. No shortcuts here.

1. Log in to Microsoft Rewards using your Microsoft account.
2. Then also log in to your Microsoft account on bing.com (just in case if not).

Yeah, both. Microsoft loves double confirmation, very aunty vibes. Once that is sorted, points actually start tracking properly.

Now for daily set points, this is where people usually mess up and then complain online.

* Enable daily set inside the extension settings.
* Allow popups for rewards.bing.

I know popups sound scary, but trust me, this one is necessary. No popup, no points. Although it usually open new tab but still makesure you allow it.

For mobile points, You can download the official Microsoft Rewards Android app.

Now the fun part. The automation. What this extension basically does is open random Bing tabs, let them load, then close them. That is it. No hacking, no shady stuff, just automated boredom. You can control everything, which I love.

* You can choose how many searches it does.
* You can set the time gap between tabs opening in seconds.
* You can even set it to 0 seconds if you want chaos and all tabs at once.
* You can choose how long before each tab closes, or set it to close as soon as it loads.

My personal setup is chill. Not too aggressive, not too slow. Feels human enough so Bing does not get suspicious like bro why are you searching so fast.

The best part, it automatically opens tabs every day when you open your browser. This is on by default. So I open Chrome or Firefox, go make chai â˜•, come back, and boom, points done.
